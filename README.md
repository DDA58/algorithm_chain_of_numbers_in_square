# AlgorithmChainOfNumbersInSquare

## Installation

Execute command to init

```bash
$ make init
```

or command to init and run

```bash
$ make init-run
```

## Usage

Execute command to run

```bash
$ make run
```

Enter amount of numbers in the side of the square(for example 3) and will get result

| index |  0  |  1  |  2  |
|-------|:---:|:---:|:---:|
| 0     |  0  |  1  |  2  |
| 1     |  7  |  8  |  3  |
| 2     |  6  |  5  |  4  |
